## 🧰 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages (deployment)

## 📌 Features

- Responsive and clean layout
- Bilingual language support (🇹🇷 Turkish / 🇬🇧 English)
- Downloadable PDF resume
- About Me, Projects, Contact sections
- Hosted with GitHub Pages
